# Aplikasi Pengelolaan Buku Perpustakaan

## Deskripsi
Aplikasi desktop berbasis Java (Swing) untuk pengelolaan data buku,
peminjaman, pengembalian, dan laporan perpustakaan.

## Fitur
- Login Admin 
- CRUD Buku
- CRUD Peminjaman & Pengembalian
- Laporan

## Database
Gunakan file `pengelolaan_buku_perpustakaan.sql`
